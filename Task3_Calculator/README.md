
# Basic Calculator

## Overview

This project is a responsive **Basic Calculator** developed using **HTML5**, **CSS3**, and **JavaScript**. It performs basic arithmetic operations through an interactive user interface while demonstrating the fundamentals of front-end web development.

The calculator features a clean design, responsive layout, and interactive functionality using JavaScript event listeners and conditional statements.

---

## Features

* User-friendly calculator interface
* Responsive layout using CSS Grid
* Display screen for input and output
* Addition operation
* Subtraction operation
* Multiplication operation
* Division operation
* Decimal number support
* Clear (C) button
* Backspace (←) functionality
* Error handling for invalid expressions
* Smooth button hover and click effects

## How to Run the Project

1. Download or clone the project files.
2. Place `index.html`, `style.css`, and `script.js` in the same folder.
3. Open `index.html` in any modern web browser.
4. Start performing calculations using the on-screen buttons.
