# Personal Portfolio Website

## Overview

This project is a responsive **Personal Portfolio Website** developed using **HTML5** and **CSS3**. It showcases my personal information, technical skills, projects, resume, and contact details in a clean and user-friendly interface.

The portfolio is designed as a beginner-friendly web development project while following responsive design principles to ensure compatibility across different screen sizes.

---

## Features

* Responsive and modern user interface
* Professional header with name and career objective
* About Me section with profile image and short biography
* Skills section highlighting technical expertise
* Projects section showcasing completed projects
* Resume download option
* Contact information with icons
* Responsive navigation bar
* Mobile-friendly design using CSS Media Queries

---

## Technologies Used

* HTML5
* CSS3

## How to Run the Project

1. Download or clone the project files.
2. Place all files in the same folder.
3. Ensure `profile.jpg` and `resume.pdf` are present in the project directory.
4. Open `index.html` in any modern web browser.
